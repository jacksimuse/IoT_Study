## 디렉터리 이름 변경하는법
만약 문제나 작성한 코드를 업로드하는데 디렉터리 이름이 잘못됬을때 간단하게 해결할 수 있습니다.
- 맨 앞에 '/'를 입력하면 디렉터리가 새로 생성됩니다.
- 맨 앞에 '../'를 입력하거나 백스페이스를 누르면 앞 폴더의 이름을 수정할 수 있습니다.
