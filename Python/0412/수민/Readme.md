# Q1 주어진 자연수가 홀수인지 짝수인지 판별해 주는 함수(is_odd)를 작성해 보자.
```python
def is_odd(number):
	if number % 2 == 0:
		print("짝수")
	else:
		print("홀수")

```

```python
input2 = input("두번째 숫자를 입력하세요:")

total = input1 + input2

```
